# 剧场假发勾织维修单

## 启动

```bash
npm install
npm start
```

默认地址：http://localhost:3909

数据保存在`data/db.json`，后续可以继续增量迭代。

## 模块说明

### 1. 看板
展示系统概览统计和演出前优先清单，快速了解假发档案总数、可演出数量、演出排期数量、维修单数量和紧急维修数量。

### 2. 演出排期（新增）

用于登记剧目演出的排期信息，关联需要使用的假发。

**功能特性：**
- 登记剧目、场次日期、角色信息
- 从已有假发档案中选择需要使用的假发
- 排期列表实时显示关联假发的当前可用状态（可演出/需要维修/紧急维修/已归还入库）
- 支持按剧目、角色搜索排期
- 支持按排期状态（已排期/待确认）筛选
- 提供"确认排期"快捷操作按钮

**排期状态说明：**
- **已排期**：排期已确认，正常安排中
- **待确认**：排期信息待核实确认

**假发状态实时显示：**
- 🟢 可演出：假发状态良好，可正常使用
- 🟡 需要维修：假发有小问题，需要维修后使用
- 🔴 紧急维修：假发问题严重，需紧急维修
- ⚫ 已归还入库：假发已归还入库

### 3. 假发档案
管理所有假发的基础信息，包括角色、剧目、发色、发网尺寸、发际线类型、存放位置、演出日期、可用状态和备注。

### 4. 维修流转
登记和跟踪假发的维修任务，包括维修类型、处理人、截止日期、状态、处理内容和结果。完成维修后会自动将关联假发标记为"可演出"状态。

### 5. 耗材台账（新增）

管理维修所需的各类耗材（勾针、蕾丝、发束、定型用品等），记录库存数量和安全库存阈值，低于安全库存的项目会自动高亮提醒。

**功能特性：**
- 登记耗材名称、库存数量、安全库存和备注
- 自动计算库存状态（库存充足 / 库存不足 / 库存告警）
- 低于安全库存的耗材卡片会以橙色或红色告警样式高亮显示
- 库存告警的耗材自动排在列表最前面，方便优先补货
- 支持按耗材名称、备注搜索
- 看板统计显示耗材总种类数和库存告警数

**库存状态说明：**
- 🟢 **库存充足**：库存数量 ≥ 安全库存
- 🟡 **库存不足**：库存数量 < 安全库存，但 ≥ 安全库存的 50%
- 🔴 **库存告警**：库存数量 < 安全库存的 50%，或库存为 0

**注意**：维修单登记时暂不自动扣减库存，需手动更新耗材台账的库存数量。

### 6. 批量导入假发档案（新增）

支持通过粘贴 CSV 文本批量导入假发档案，导入前可预览识别结果，标出缺失必填字段的行，确认后仅导入合法数据。

**功能特性：**
- 粘贴 CSV 文本即可解析，无需上传文件
- 自动识别中文字段名（角色、剧目、发色、存放位置、演出日期等）
- 预览阶段高亮显示缺少必填字段的行和单元格
- 显示解析统计：有效行数、缺失行数、总行数
- 确认导入时只写入合法行数据
- 导入结果展示：成功数量、失败数量、失败行详情
- 可选字段自动填充默认值（发网尺寸默认 M、发际线默认普通前网、状态默认可演出）

**必填字段（带 * 号）：**
- 角色 *
- 剧目 *
- 发色 *
- 存放位置 *
- 演出日期 *

**可选字段：**
- 发网尺寸（默认 M）
- 发际线类型（默认 普通前网）
- 可用状态（默认 可演出）
- 备注

**示例 CSV 数据：**

```csv
角色,剧目,发色,发网尺寸,发际线类型,存放位置,演出日期,可用状态,备注
花仙子,绿野仙踪,金色长卷,M,手勾蕾丝前额,C柜-02,2026-07-15,可演出,新制作
女巫,绿野仙踪,黑色大卷,L,普通前网,C柜-03,2026-07-15,可演出,
国王,哈姆雷特,深棕短波,S,,D柜-01,2026-08-01,,可选字段为空不影响
王后,哈姆雷特,,M,手勾蕾丝前额,D柜-02,2026-08-01,可演出,缺少必填发色
侍卫,天鹅湖,白色,M,普通前网,,2026-07-20,可演出,缺少必填存放位置
```

**使用流程：**
1. 点击顶部"批量导入"标签页
2. 将 CSV 文本粘贴到文本框中（第一行为表头）
3. 点击"解析预览"按钮查看识别结果
4. 检查预览表格：红色背景行表示该行缺失必填字段，红色单元格标出具体缺失位置
5. 确认无误后点击"确认导入"按钮（仅导入有效行）
6. 查看导入结果统计，如有失败行可查看详情

### 演出前检查模板配置（新增）

支持为特定剧目或假发类型配置额外检查项模板。默认使用通用检查项，匹配到模板时会自动添加额外检查项。

**配置结构说明**：

模板配置位于 `project.config.js` 的 `checkTemplates` 数组中，支持以下匹配方式：

1. **按剧目匹配**：使用 `show` 字段指定剧目名称
2. **按假发发际线类型匹配**：使用 `hairlineType` 字段指定发际线类型（如：手勾蕾丝前额、普通前网）

**模板字段说明**：
```javascript
{
  id: 'show-midnight-garden',      // 模板唯一ID
  name: '午夜花园 专属检查',        // 模板显示名称
  show: '午夜花园',                 // 剧目匹配（可选）
  hairlineType: '手勾蕾丝前额',     // 假发类型匹配（可选）
  extraItems: ['头饰固定', '蕾丝边缘', '发缝遮盖'],  // 额外检查项
  description: '模板说明'
}
```

**匹配优先级**：
- 剧目模板优先于假发类型模板
- 多个模板匹配时，额外检查项会自动去重合并
- 无匹配模板时使用通用检查项

**内置示例模板**：
1. `午夜花园 专属检查` - 匹配剧目"午夜花园"，添加：头饰固定、蕾丝边缘、发缝遮盖
2. `蕾丝前额假发 专属检查` - 匹配发际线类型"手勾蕾丝前额"，添加：蕾丝边缘、发缝遮盖、胶层状态
